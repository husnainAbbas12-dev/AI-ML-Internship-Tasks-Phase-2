## 📌 Overview

This repository contains a collection of projects completed during my internship, focusing on **Machine Learning, NLP, and Large Language Models (LLMs)**.

Each project demonstrates real-world applications such as:

* Automating customer support workflows
* Predicting customer churn
* Building context-aware AI chatbots

---

## 📂 Projects

### 🔹 1. Auto Tagging Support Tickets Using LLM

Automates classification of customer support tickets using **zero-shot learning**.

#### ✨ Highlights

* Zero-shot classification (no labeled training data required)
* Uses transformer-based models
* Multi-category tagging (billing, refund, technical issues)

#### 🛠️ Tech Stack

`Python` `Transformers` `PyTorch`

---

### 🔹 2. Customer Churn Prediction System

Predicts whether customers are likely to leave a service using machine learning.

#### ✨ Highlights

* End-to-end ML pipeline
* Feature engineering & preprocessing
* Hyperparameter tuning with GridSearchCV

#### 🛠️ Tech Stack

`Python` `Pandas` `NumPy` `Scikit-learn`

---

### 🔹 3. Context-Aware Chatbot

A chatbot that answers questions based on uploaded documents (PDF/text).

#### ✨ Highlights

* Context-aware responses using document retrieval
* Text chunking for efficient processing
* Integration with LangChain

#### 🛠️ Tech Stack

`Python` `LangChain` `Transformers` `PyPDF`
